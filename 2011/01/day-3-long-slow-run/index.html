Day 3 - Long (Slow?) Run!

Jan 3rd, 2011

Bloody dark this morning, which made the initial cross country parts hard work.  It didn’t help, that I merrily ran out in my road trainers instead of my trail ones, so I was slipping round some of the early fields like bambi.

Adam set a hard pace; I was suffering from too many mince pies over christmas!  I found the first 5 miles hardwork and took a few walk breaks.  I also forgot what a b*stard the hill is at the start - I sounded like darth vader as I ran/stumbled up it!

Excellent on-in should have run with a negative split which is a first!

Stats:

Distance: 7.92 mi
Duration: 1:24:18
Avg. Pace: 10:39 min/mi
